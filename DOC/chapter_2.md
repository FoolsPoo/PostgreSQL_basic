## Basic Command

List All Databases: “\l”.

Access or Switch a Database: “\c db_name”.

List All Tables: “\dt”.

Describe All Tables: “\d”.

Describe a Specific Table: “\d tab_name”.

List All Schemas: “\dn”.

List All Views: “\dv”.

List All Functions: “\df”.

List All Users: “\du”.

Quit psql: “\q” or "Ctrl+Z".

สามารถดูคำสั่งเพิ่มเติมโดยใช้คำสั่ง "\?"
